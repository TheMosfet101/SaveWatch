# SaveWatch
A device used for the benefit of elderly people, which alert others and themselves whenever they fall or get too close towards an oncoming object. It uses Arduino Nano as the main board, a ADXL335 accelerometer,VL53L0X time of flight sensor,I2L LCD Adapter with an LCD display screen, and other passive/active components.
